# AI Insight Copilot

A lightweight AI-style analytics copilot that turns business questions into SQL-style logic, plain-English summaries, visual breakdowns, and recommended next actions.

## Features

- Question-to-insight workflow for common analytics questions
- Generated SQL-style query logic for transparency
- Revenue, channel, region, product, satisfaction, and trend analysis
- Browser dashboard with canvas charts
- No external dependencies

## Run Locally

```bash
npm start
```

Open `http://localhost:3000`.

## Test

```bash
npm test
```

## Example Questions

- Which region has the highest revenue?
- Which channel is performing best?
- Show revenue trend over time.
- Which product has the most revenue?

## Stack

Node.js, JavaScript, CSV analytics, HTML, CSS, Canvas.
